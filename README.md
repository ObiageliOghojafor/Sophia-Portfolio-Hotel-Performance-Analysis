## Introduction
---
This project analyzes key hotel metrics using Power BI.The goal is to transform raw hotel data into clear insights that support better business decisions. Key areas explored include total bookngs, total revenue, avarage daily rate(ADR), markert segment and weekday vs. weekend patterns.The dashboard uses data modeling, Power Query transformations, and DAX measures to identify booking patterns, guest demogrphic and performance opportunities within the hospitality industry.
